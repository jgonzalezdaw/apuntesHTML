Etiquetas básicas

Para marcar los encabezados usaremos <h1>, <h2>, <h3>, <h4>, <h5> o <h6>

Para marcar un párrafo usaremos <p>

Para organizar el documento podemos usar <header>, <main> y <footer>. También tenemos las etiquetas <article>, <section> y <aside>

Para crear una barra de navegación usaremos <nav> y enlaces

Listas

  • Para listas ordenadas usaremos <ol>
  • Para listas no ordenadas usaremos <ul>
  • Para marcar cada ítem de la lista usaremos <li>
